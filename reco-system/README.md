\# Reco-System 🎬📚



!\[.NET CI](https://github.com/Milenalp07/reco-system/actions/workflows/dotnet-tests.yml/badge.svg)



A Book and Movie Recommendation Platform built with ASP.NET Core MVC (.NET 8), Entity Framework Core, SQLite, and the TMDB API.



\## Features

\- User authentication with role-based access (Admin, User, Premium)

\- Book library with CRUD operations and related book suggestions

\- Movie discovery via TMDB API with similar movies

\- Homepage carousel with featured movies

\- Subscription plans (Free and Premium)

\- CI/CD pipeline with GitHub Actions and xUnit tests



\## Tech Stack

\- ASP.NET Core MVC (.NET 8)

\- Entity Framework Core + SQLite

\- ASP.NET Core Identity

\- TMDB API

\- Tailwind CSS

\- xUnit + GitHub Actions
